# ARTIST_OS Skill: Spotify & Socials Integration

## 1. API Integration Matrix
- **Spotify for Artists**: Stream counts, listener demographics, core fan vs. fan ratios.
- **YouTube/Google Data**: Channel link binding, video analytics (bypassing ad-blocker limitations).
- **Social Graph**: Instagram, TikTok, Facebook engagement metrics.

## 2. The "Fan Portal" Sync
- **Logic**: Aggregates cross-platform data to segment audience into "Core" and "Broad" fans.
- **Frequency**: Real-time polling with every dashboard refresh.
- **Metric Mapping**: Core fans match YouTube subscribers/Spotify followers in high-intent categories.

## 3. Career KPIs
- **Market Penetration**: Tracked via cross-platform reach.
- **Fan Engagement**: Interaction velocity across all linked socials.
- **Sync Status**: Displayed as "SYNC_ACTIVE" or "ERROR" on Artist Dashboard.
