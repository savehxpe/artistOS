Purpose
A high-velocity, low-friction command center strictly engineered for the artist. It removes administrative noise, visually prioritizing the current rollout phase, urgent legal sign-offs, and critical fan conversion data.
Dashboard Architecture (Bento Grid)
The interface is constrained to a 4-quadrant grid, prioritizing functionality over aesthetic embellishment
:
MISSION_STATUS (Hero): High-contrast progress bar tracking the active campaign phase (Pre-Release, Release Week, Post-Release). Auto-populates the 54-item task stack post-diagnostic.
URGENT_STACK (Action): The only quadrant utilizing #FFFF00 (Vibrant Yellow) highlights
. Displays critical bottlenecks requiring immediate digital sign-off (e.g., manager approvals, split-sheets).
VAULT_PULSE (Asset): Horizontal feed of newly ingested masters, stems, and visuals. All audio streaming utilizes secure, watermarked previews.
GROWTH_METRICS (KPIs): Real-time API sync displaying the "Follower-to-Listener Conversion Rate" across TikTok and Spotify
.
Dynamic Logic (SYSTEM_GLUE)
Diagnostic Gateway: If no active project is detected, the UI locks the grid and forces a 5-Question Onboarding Diagnostic to instantly build the rollout timeline.
AI_STORY_GEN Integration: Tapping an asset within the VAULT_PULSE instantly routes the audio to the AI Copilot, generating high-converting, 3-second social hooks
 and SEO-optimized captions.
Interaction & Aesthetic Standard
Palette: Strict monochrome base. Zero extraneous colors.
Tactile Feedback: Every interaction (tap, swipe, approval) triggers a mandatory 0.95 active scale transition, confirming the action instantly without distracting animations.
Temporal Lock: All data displays, predictive timelines, and algorithmic triggers are strictly bound to the system master date: March 27, 2026.
